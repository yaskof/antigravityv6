# SiparişMatik (antigravityv6)

SiparişMatik, çok kanallı restoran siparişlerini tek panelde toplayan ve kurye atamasını yöneten bir yönetim paneli örneğidir.

## Kurulum

```bash
npm install
```

## Uygulamayı Görüntüleme

Frontend geliştirme sunucusunu çalıştırın:

```bash
npm run dev
```

Ardından tarayıcıdan şu adrese gidin:

```
http://localhost:5173
```

> Not: Farklı bir port kullanılıyorsa terminal çıktısındaki URL’yi takip edin.

## Backend (Webhook Gateway)

Webhook gateway’i çalıştırmak için:

```bash
npm run server
```

Varsayılan adres:

```
http://localhost:5050/health
```
